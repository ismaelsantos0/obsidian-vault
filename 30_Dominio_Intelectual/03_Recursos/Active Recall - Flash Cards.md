## 🏦 AFO #flashcards/afo

Qual a sigla dos 3 instrumentos orçamentários? :: **PPA → LDO → LOA**
Prazo de envio da LDO ao Legislativo? :: **15 de abril**
Prazo de envio da LOA ao Legislativo? :: **31 de agosto**
Qual lei NÃO pode deixar de ser aprovada antes do recesso? :: **LDO**
Quais são as etapas da Despesa Pública? :: **Empenho → Liquidação → Pagamento**
O que é o Empenho? :: Reserva de crédito. Compromete a dotação orçamentária.
O que é a Liquidação? :: Verificação do direito do credor (ateste da entrega).
Qual modalidade de empenho para energia elétrica? :: **Estimativa** (valor incerto)
Qual modalidade de empenho para um contrato parcelado? :: **Global**
Quais são os estágios da receita? :: **Previsão → Lançamento → Arrecadação → Recolhimento**
Qual a diferença entre Receita Originária e Derivada? :: Originária = patrimônio próprio do Estado. Derivada = poder de império (tributos, multas).
Quais são as 3 ages dos arquivos? :: **Corrente → Intermediário → Permanente**
O arquivo permanente pode ser destruído? :: **Jamais.** Tem valor histórico/probatório.
Qual o princípio arquivístico que proíbe misturar documentos de origens diferentes? :: **Princípio da Proveniência (Respeito aos Fundos)**
Qual gênero documental inclui fotos e mapas? :: **Iconográfico**

## ⚖️ Direito Constitucional e Administrativo #flashcards/direito

Quais são os princípios LIMPE? :: **L**egalidade, **I**mpessoalidade, **M**oralidade, **P**ublicidade, **E**ficiência
Qual princípio foi acrescentado pela EC 19/1998? :: **Eficiência**
O particular pode fazer o que a lei não proíbe. E a Administração? :: **Só pode fazer o que a lei expressamente autoriza** (Legalidade estrita)
O que diferencia autarquias de empresas públicas quanto à criação? :: Autarquias: criadas **por lei**. Empresas públicas: criadas por lei **autorizadora**.
Segurança Pública é Uti Universi ou Uti Singuli? :: **Uti Universi** (indivisível, pago por imposto)
Segurança Pública é delegável? :: **Não. É indelegável.**
Quais os atributos do ato administrativo (TIPPA)? :: **T**ipicidade, **I**mperatividade, **P**resumção de legitimidade, **P**revisão, **A**uto-executoriedade
Quem pode anular um ato administrativo ilegal? :: A própria **Adm. Pública** ou o **Judiciário**
Quem pode revogar um ato? :: Apenas a **Adm. Pública** (por conveniência/oportunidade)
Revogação tem efeito ex tunc ou ex nunc? :: **Ex nunc** (só para frente)
Anulação tem efeito ex tunc ou ex nunc? :: **Ex tunc** (retroativo)
Qual modalidade de licitação para bens e serviços comuns, qualquer valor? :: **Pregão**
O que o Diálogo Competitivo acrescentou? :: Nova modalidade criada pela **Lei 14.133/21**
Quem exerce o Controle Externo da Adm. Federal? :: **Congresso Nacional**, com auxílio do **TCU**

## 📜 Processo Legislativo (CF/88 + CE-RR) #flashcards/proclep

Quais as espécies normativas do Art. 59 CF/88? :: PEC, LC, LO, LD, MP, DL, Resoluções
Qual quórum para aprovar LC? :: **Maioria Absoluta**
Qual quórum para aprovar LO? :: **Maioria Simples**
Qual quórum para aprovar PEC? :: **3/5** dos membros, em **2 turnos**
Quantos deputados tem a ALE-RR? :: **24**
Maioria Absoluta na ALE-RR = quantos deputados? :: **13** (metade + 1 do total de 24)
3/5 na ALE-RR = quantos deputados? :: **≥ 15**
Prazo para o Governador vetar? :: **15 dias úteis**
O que acontece se o Governador não se manifestar em 15 dias? :: **Sanção Tácita** (projeto vira lei)
Prazo para a ALE votar o veto? :: **30 dias**
Decretos Legislativos passam pelo Governador? :: **Não.** São promulgados pela Mesa da ALE-RR.
Quórum para a PEC na CE-RR? :: **3/5 (15 deputados)** em **2 turnos** com intervalo mínimo de **10 dias**

## 🏛️ Regimento Interno e Estatuto ALE-RR #flashcards/regimento

Quais os períodos da Sessão Legislativa Ordinária? :: **15/02 a 30/06** e **01/08 a 22/12**
Qual comissão é obrigatória para TODA proposição? :: **CCJ** (Constituição e Justiça)
Prazo de parecer em urgência? :: **10 dias**
Prazo de parecer em prioridade? :: **12 dias**
Prazo de parecer ordinário? :: **18 dias**
Quem é o 2º na linha de sucessão do Governador? :: **Presidente da ALE-RR**
Mandato da Mesa Diretora? :: **2 anos**, permitida **1 recondução**
Após quantos anos o servidor efetivo é estável? :: **3 anos** (estágio probatório)
Quantos dias de férias tem o servidor? :: **30 dias** anuais
O que é abandono de cargo? :: Ausência injustificada por mais de **30 dias consecutivos**
Qual penalidade para abandono de cargo? :: **Demissão**

## 🗺️ Roraima #flashcards/roraima

Quando Roraima virou Território Federal? :: **1943** (Decreto-Lei 5.812)
Quando Roraima virou Estado? :: **1988** (CF/88, Art. 14 ADCT)
Qual o ponto mais alto do Brasil? :: **Monte Roraima** (Serra do Pacaraima) — 2.875m
Monte Roraima é tríplice fronteira com quais países? :: **Brasil, Venezuela e Guiana**
Qual o principal rio de Roraima? :: **Rio Branco** (afluente do Rio Negro)
Qual a única capital brasileira ao Norte do equador? :: **Boa Vista**
Quantos municípios tem Roraima? :: **15**
Qual o principal ecossistema do centro de RR? :: **Lavrado** (savana)
Roraima faz fronteira com quais países? :: **Venezuela** (N/NO) e **Guiana** (L/NE)
Qual o ciclo econômico que impulsionou a colonização? :: **Ciclo do Gado** (pecuária no lavrado)
