# 💻 Padrões e Convenções de Código

## 🧬 Nomenclatura
*   **CamelCase** para classes e componentes React.
*   **snake_case** para chaves do banco de dados.
*   **kebab-case** para rotas de API.

## 🛡️ Segurança
*   LGPD: Encriptação de dados sensíveis em repouso.
*   Log de Auditoria: Toda alteração deve ser registrada com ID do autor.

## 📦 Git Workflow
*   `main`: Produção.
*   `develop`: Integração.
*   `feature/*`: Novas funcionalidades.
